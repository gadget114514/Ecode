# Ecode Dokumentation

Ecode ist ein leichtgewichtiger, nativer Win32-Texteditor, der auf Leistung und Einfachheit ausgelegt ist.

## Funktionen

### Bearbeiten
- **Syntax-Hervorhebung**: Grundlegende Hervorhebung für C/C++, Java, JavaScript, Python und mehr.
- **Automatische Einrückung**: Intelligente Einrückung basierend auf der Sprache.

### Navigation
- **Emacs Tastenkombinationen**: Verwenden Sie Standard-Emacs-Shortcuts für die Navigation (`C-n`, `C-p`, etc.).
- **Gehe zu Zeile**: Springen Sie zu bestimmten Zeilen mit `Ctrl+G`.
- **Suchen/Ersetzen**: Suchen und Ersetzen von Text mit `Ctrl+F` und `Ctrl+H`.

### Dateiverwaltung
- **In Dateien suchen**: Verzeichnisse rekursiv durchsuchen (`Ctrl+Shift+F`).
- **Zuletzt geöffnete Dateien**: Schneller Zugriff auf zuletzt geöffnete Dateien.

### Extras
- **Shell-Integration**: Shell-Befehle direkt im Editor-Puffer ausführen.
- **Scripting**: Funktionalität mit JavaScript (quickjs) erweitern.
- **Makro-Galerie**: Vordefinierte Makros ausführen.

## Konfiguration
Einstellungen können über das Menü **Konfiguration > Einstellungen** angepasst werden.
- **Design**: Ändern Sie das Farbschema.
- **Shell-Kodierung**: Wechseln Sie zwischen UTF-8 und Shift-JIS für die Shell-Ausgabe.

## Fehlerbehebung
Wenn Probleme auftreten, überprüfen Sie den Puffer `*Messages*` auf Protokolle.
