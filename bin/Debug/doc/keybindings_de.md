# Ecode Tastenkombinationen

## Dateioperationen
| Aktion | Tastenkombination |
| :--- | :--- |
| Datei öffnen | `C-x C-f` |
| Datei speichern | `C-x C-s` |
| Speichern unter | `C-x C-w` |
| Puffer schließen | `C-x k` |
| Beenden | `C-x C-c` |

## Navigation
| Aktion | Tastenkombination |
| :--- | :--- |
| Vorwärts bewegen (rechts) | `C-f` |
| Rückwärts bewegen (links) | `C-b` |
| Nächste Zeile (unten) | `C-n` |
| Vorherige Zeile (oben) | `C-p` |
| Zeilenanfang | `C-a` |
| Zeilenende | `C-e` |
| Seite nach unten | `C-v` |
| Seite nach oben | `M-v` (Alt+V) |

## Bearbeiten
| Aktion | Tastenkombination |
| :--- | :--- |
| Suchen | `C-s` |
| Ausschneiden (Region) | `C-w` |
| Einfügen (Yank) | `C-y` |
| Kopieren | `C-c` |
| Rückgängig | `C-z` |
| Zeichen löschen | `C-d` |
| Zeile löschen (Kill) | `C-k` |

## Sonstiges
| Aktion | Tastenkombination |
| :--- | :--- |
| Befehl ausführen (Minibuffer) | `M-x` (Alt+X) |
| Abbrechen / Beenden | `C-g` |
| In Dateien suchen | `C-S-f` |
| Vergrößern | `Ctrl++` |
| Verkleinern | `Ctrl+-` |

*Hinweis: `C-x` steht für das Control+X-Präfix. `M-v` steht für Alt+V.*
