# Documentación de Ecode

Ecode es un editor de texto nativo Win32 ligero, diseñado para el rendimiento y la simplicidad.

## Características

### Edición
- **Resaltado de sintaxis**: Resaltado básico para C/C++, Java, JavaScript, Python y más.
- **Auto-sangría**: Sangría inteligente basada en el lenguaje.

### Navegación
- **Atajos de teclado de Emacs**: Use atajos estándar de Emacs para la navegación (`C-n`, `C-p`, etc.).
- **Ir a la línea**: Salte a líneas específicas con `Ctrl+G`.
- **Buscar/Reemplazar**: Busque y reemplace texto con `Ctrl+F` y `Ctrl+H`.

### Gestión de archivos
- **Buscar en archivos**: Busque en directorios de forma recursiva (`Ctrl+Shift+F`).
- **Archivos recientes**: Acceda rápidamente a archivos abiertos recientemente.

### Herramientas
- **Integración de Shell**: Ejecute comandos de shell directamente en el búfer del editor.
- **Scripting**: Extienda la funcionalidad con JavaScript (quickjs).
- **Galería de macros**: Ejecute macros predefinidas.

## Configuración
Los ajustes se pueden ajustar a través del menú **Config > Configuración**.
- **Tema**: Cambie el esquema de colores.
- **Codificación de Shell**: Alterne entre UTF-8 y Shift-JIS para la salida de shell.

## Solución de problemas
Si encuentra problemas, consulte el búfer `*Messages*` para ver los registros.
