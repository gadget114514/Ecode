# Raccourcis clavier d'Ecode

## Opérations de fichier
| Action | Raccourci |
| :--- | :--- |
| Ouvrir le fichier | `C-x C-f` |
| Enregistrer le fichier | `C-x C-s` |
| Enregistrer sous | `C-x C-w` |
| Fermer le tampon | `C-x k` |
| Quitter | `C-x C-c` |

## Navigation
| Action | Raccourci |
| :--- | :--- |
| Avancer (droite) | `C-f` |
| Reculer (gauche) | `C-b` |
| Ligne suivante (bas) | `C-n` |
| Ligne précédente (haut) | `C-p` |
| Début de ligne | `C-a` |
| Fin de ligne | `C-e` |
| Page suivante | `C-v` |
| Page précédente | `M-v` (Alt+V) |

## Édition
| Action | Raccourci |
| :--- | :--- |
| Rechercher | `C-s` |
| Couper (région) | `C-w` |
| Coller (Yank) | `C-y` |
| Copier | `C-c` |
| Annuler | `C-z` |
| Supprimer le caractère | `C-d` |
| Supprimer la ligne (Kill) | `C-k` |

## Autres
| Action | Raccourci |
| :--- | :--- |
| Exécuter la commande (Minibuffer) | `M-x` (Alt+X) |
| Annuler / Quitter | `C-g` |
| Rechercher dans les fichiers | `C-S-f` |
| Zoom avant | `Ctrl++` |
| Zoom arrière | `Ctrl+-` |

*Note: `C-x` indique le préfixe Control+X. `M-v` indique Alt+V.*
