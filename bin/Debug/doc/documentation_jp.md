# Ecode ドキュメント

Ecodeは、パフォーマンスとシンプルさを重視して設計された軽量なネイティブWin32テキストエディタです。

## 機能

### 編集
- **シンタックスハイライト**: C/C++、Java、JavaScript、Pythonなどの基本的なシンタックスハイライト。
- **自動インデント**: 言語に基づいたスマートなインデント。

### ナビゲーション
- **Emacsキーバインド**: 標準的なEmacsショートカット (`C-n`, `C-p` など) を使用可能。
- **指定行へ移動**: `Ctrl+G` で特定の行へジャンプ。
- **検索/置換**: `Ctrl+F` と `Ctrl+H` でテキストの検索と置換。

### ファイル管理
- **ファイル内検索**: ディレクトリを再帰的に検索 (`Ctrl+Shift+F`)。
- **最近使ったファイル**: 最近開いたファイルに素早くアクセス。

### ツール
- **シェル統合**: エディタバッファ内で直接シェルコマンドを実行。
- **スクリプト**: JavaScript (quickjs) で機能を拡張。
- **マクロギャラリー**: 定義済みマクロを実行。

## 設定
設定は **設定 > 設定** メニューから調整できます。
- **テーマ**: 配色の変更。
- **シェルエンコーディング**: シェル出力の UTF-8 と Shift-JIS の切り替え。

## トラブルシューティング
問題が発生した場合は、`*Messages*` バッファのログを確認してください。
