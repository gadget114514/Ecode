# Ecode キーバインド

## ファイル操作
| アクション | ショートカット |
| :--- | :--- |
| ファイルを開く | `C-x C-f` |
| 保存 | `C-x C-s` |
| 名前を付けて保存 | `C-x C-w` |
| バッファを閉じる | `C-x k` |
| 終了 | `C-x C-c` |

## ナビゲーション
| アクション | ショートカット |
| :--- | :--- |
| 前に進む (右) | `C-f` |
| 後ろに戻る (左) | `C-b` |
| 次の行 (下) | `C-n` |
| 前の行 (上) | `C-p` |
| 行頭へ移動 | `C-a` |
| 行末へ移動 | `C-e` |
| ページダウン | `C-v` |
| ページアップ | `M-v` (Alt+V) |

## 編集
| アクション | ショートカット |
| :--- | :--- |
| 検索 | `C-s` |
| 切り取り (領域) | `C-w` |
| 貼り付け (ヤンク) | `C-y` |
| コピー | `C-c` |
| 元に戻す | `C-z` |
| 文字削除 | `C-d` |
| 行削除 (キル) | `C-k` |

## その他
| アクション | ショートカット |
| :--- | :--- |
| コマンド実行 (ミニバッファ) | `M-x` (Alt+X) |
| キャンセル / 終了 | `C-g` |
| ファイル内検索 | `C-S-f` |
| 拡大 | `Ctrl++` |
| 縮小 | `Ctrl+-` |

*注: `C-x` は Control+X プレフィックスを示します。 `M-v` は Alt+V を示します。*
