# Ecode Documentation

Ecode is a lightweight, native Win32 text editor designed for performance and simplicity.

## Features

### Editing
- **Sytax Highlighting**: Basic syntax highlighting for C/C++, Java, JavaScript, Python, and more.
- **Auto-indentation**: Smart indentation based on language.
- **Multi-cursor**: (Planned) Support for multiple carets.

### Navigation
- **Emacs Keybindings**: Use standard Emacs shortcuts for navigation (`C-n`, `C-p`, etc.).
- **Go to Line**: Jump to specific lines with `Ctrl+G`.
- **Find/Replace**: Search and replace text with `Ctrl+F` and `Ctrl+H`.

### File Management
- **Find in Files**: Recursively search directories (`Ctrl+Shift+F`).
- **Recent Files**: Quickly access recently opened files.

### Tools
- **Shell Integration**: Run shell commands directly in the editor buffer.
- **Scripting**: Extend functionality with JavaScript (quickjs).
- **Macro Gallery**: Run predefined macros.

## Configuration
Settings can be adjusted via the **Config > Settings** menu.
- **Theme**: Change the color scheme.
- **Shell Encoding**: Switch between UTF-8 and Shift-JIS for shell output.

## Troubleshooting
If you encounter issues, check the `*Messages*` buffer for logs.
