# Ecode Keybindings

## File Operations
| Action | Shortcut |
| :--- | :--- |
| Open File | `C-x C-f` |
| Save File | `C-x C-s` |
| Save As | `C-x C-w` |
| Close Buffer | `C-x k` |
| Exit | `C-x C-c` |

## Navigation
| Action | Shortcut |
| :--- | :--- |
| Move Forward (Right) | `C-f` |
| Move Backward (Left) | `C-b` |
| Move Next Line (Down) | `C-n` |
| Move Previous Line (Up) | `C-p` |
| Move Beginning of Line | `C-a` |
| Move End of Line | `C-e` |
| Page Down | `C-v` |
| Page Up | `M-v` (Alt+V) |

## Editing
| Action | Shortcut |
| :--- | :--- |
| Find | `C-s` |
| Cut (Region) | `C-w` |
| Paste (Yank) | `C-y` |
| Copy | `C-c` |
| Undo | `C-z` |
| Delete Char | `C-d` |
| Kill Line | `C-k` |

## Other
| Action | Shortcut |
| :--- | :--- |
| Execute Command (Minibuffer) | `M-x` (Alt+X) |
| Cancel / Quit | `C-g` |
| Find in Files | `C-S-f` |
| Zoom In | `Ctrl++` |
| Zoom Out | `Ctrl+-` |

*Note: `C-x` indicates the Control+X prefix. `M-v` indicates Alt+V.*
