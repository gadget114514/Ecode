# Atajos de teclado de Ecode

## Operaciones de archivo
| Acción | Atajo |
| :--- | :--- |
| Abrir archivo | `C-x C-f` |
| Guardar archivo | `C-x C-s` |
| Guardar como | `C-x C-w` |
| Cerrar búfer | `C-x k` |
| Salir | `C-x C-c` |

## Navegación
| Acción | Atajo |
| :--- | :--- |
| Mover adelante (derecha) | `C-f` |
| Mover atrás (izquierda) | `C-b` |
| Línea siguiente (abajo) | `C-n` |
| Línea anterior (arriba) | `C-p` |
| Inicio de línea | `C-a` |
| Fin de línea | `C-e` |
| Avanzar página | `C-v` |
| Retroceder página | `M-v` (Alt+V) |

## Edición
| Acción | Atajo |
| :--- | :--- |
| Buscar | `C-s` |
| Cortar (región) | `C-w` |
| Pegar (Yank) | `C-y` |
| Copiar | `C-c` |
| Deshacer | `C-z` |
| Eliminar carácter | `C-d` |
| Cortar línea (Kill) | `C-k` |

## Otros
| Acción | Atajo |
| :--- | :--- |
| Ejecutar comando (Minibuffer) | `M-x` (Alt+X) |
| Cancelar / Salir | `C-g` |
| Buscar en archivos | `C-S-f` |
| Aumentar zoom | `Ctrl++` |
| Disminuir zoom | `Ctrl+-` |

*Nota: `C-x` indica el prefijo Control+X. `M-v` indica Alt+V.*
