# Documentation d'Ecode

Ecode est un éditeur de texte natif Win32 léger, conçu pour la performance et la simplicité.

## Caractéristiques

### Édition
- **Coloration syntaxique**: Coloration de base pour C/C++, Java, JavaScript, Python et plus.
- **Auto-indentation**: Indentation intelligente basée sur le langage.

### Navigation
- **Raccourcis clavier Emacs**: Utilisez les raccourcis standard d'Emacs pour la navigation (`C-n`, `C-p`, etc.).
- **Aller à la ligne**: Passez à des lignes spécifiques avec `Ctrl+G`.
- **Rechercher/Remplacer**: Recherchez et remplacez du texte avec `Ctrl+F` et `Ctrl+H`.

### Gestion des fichiers
- **Rechercher dans les fichiers**: Recherchez dans les répertoires de manière récursive (`Ctrl+Shift+F`).
- **Fichiers récents**: Accédez rapidement aux fichiers récemment ouverts.

### Outils
- **Intégration du Shell**: Exécutez des commandes shell directement dans le tampon de l'éditeur.
- **Scripting**: Étendez les fonctionnalités avec JavaScript (quickjs).
- **Galerie de macros**: Exécutez des macros prédéfinies.

## Configuration
Les paramètres peuvent être ajustés via le menu **Configuration > Paramètres**.
- **Thème**: Changez le jeu de couleurs.
- **Codage du Shell**: Basculez entre UTF-8 et Shift-JIS pour la sortie du shell.

## Dépannage
Si vous rencontrez des problèmes, consultez le tampon `*Messages*` pour les journaux.
